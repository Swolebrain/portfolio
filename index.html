Frogger Game Using ES6

Uses class syntax and live reload dev server

This project was part of the Front End Web Developer Nanodegree at Udacity. In its original version, it used pseudoclassical prototypal classes and inheritance. It has since been ammended to use ES6 classes, ES6 import/export syntax, and has a hot-reload-enabled dev server (budo) which uses watchify with the babelify transform and the es2015 babel preset to transpile and run.

JavaScript Exercise Grader

Pure jQuery and live reload dev server

This is a tool aimed at helping students get the practice they need in order to really learn to code. I'm developing it because I feel most free coding resources are too light on the drilling. I feel students would benefit from simpler, shorter exercises that can be completed in one or two minutes each. This project has a Node.JS RESTful back-end and a completely decoupled front-end.
